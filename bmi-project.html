Mini Project – Hydration and Temperature Detector

Objective: To design and implement a low-cost, real-time body monitoring system using the ESP32 Dev Board that accurately measures body temperature using a DS18B20 sensor and estimates hydration levels via a capacitive moisture sensor. The data is visualized on a compact OLED display, enhancing personal health tracking and biomedical research.

Components Used:

  • Hardware: ESP32 Dev Board, DS18B20 Temperature Sensor, Capacitive Moisture Sensor, 0.96” OLED Display, Breadboard, Jumper Wires
  • Software: Arduino IDE, Blynk IoT App
  • Libraries: OneWire, DallasTemperature, Adafruit SSD1306, Adafruit GFX, Blynk

Abstract:

This system continuously monitors body temperature and hydration. The DS18B20 sensor transmits temperature data via 1-Wire protocol, while the moisture sensor provides analog voltage based on skin hydration. All readings are displayed on an OLED screen and sent to the Blynk mobile app through ESP32's Wi-Fi connectivity.

Working Principle:

  • The ESP32 reads sensor data and displays it on the OLED using I2C.
  • Temperature is detected using the DS18B20 sensor with a 4.7kΩ pull-up resistor.
  • Hydration levels are derived from analog signals from the capacitive moisture sensor.
  • Data is simultaneously sent to the Blynk app for mobile access.

Applications:

  • Personal health monitoring
  • Fitness tracking
  • Biomedical research
